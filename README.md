# 🥗 Daily Meal Planner

A simple, no-nonsense **vegetarian meal planner + workout generator** built with HTML, Tailwind CSS, and vanilla JavaScript.

Designed for **South Indian diet preferences**, with a focus on:
- High protein
- Portion control
- Fat loss consistency

---

## 🚀 Features

### 🍽 Meal Generator
- Random **Breakfast, Lunch, Dinner**
- Based on real, repeatable South Indian meals
- Includes portion sizes (no guessing)

### ☕ Snack Suggestions
- Controlled quantities
- High-protein focused options

### 🏃 Workout Generator
- Simple daily exercises (home-friendly)
- No equipment dependency

### 🎥 Workout Videos
- Embedded YouTube workouts
- Removes decision fatigue

### 💬 Motivation
- Random quote fetched from API

---

## 🧠 How to Use

1. Open the app
2. Click **"Generate Today’s Meal"**
3. Follow EXACTLY — no substitutions
4. Execute the workout shown
5. Repeat daily

---

## ⚠️ Rules (Important)

### 1. Don’t regenerate multiple times
Bad:
- Clicking until you see your “favorite” meal

Good:
- First plan = final plan

---

### 2. Portion control matters
- Rice → **1 cup (150–180g cooked)**
- Chapati → **2 max**
- Chutney → **1 tbsp only**

---

### 3. Protein is non-negotiable
Every meal includes:
- Tofu / Soy / Skyr / Curd / Paneer

If you skip protein → expect hunger + cravings.

---

### 4. Snacks are controlled
- Not free eating
- Follow exact quantities
